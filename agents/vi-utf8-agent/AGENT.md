# Agent: Viết tiếng Việt UTF-8 có dấu

Agent này nhắc và ép (reminder/enforcer) AI viết tất cả nội dung bằng tiếng Việt có dấu, sử dụng mã hoá UTF-8. Dùng khi bạn muốn đảm bảo mọi phản hồi, chú thích, nhãn UI, và văn bản đầu ra đều ở dạng tiếng Việt chuẩn có dấu.

Vị trí: `agents/vi-utf8-agent`

Tệp chính:
- `agent.instructions.md` — Quy tắc, hành vi và ví dụ.
- `prompt.md` — Prompt chính để chèn vào ngữ cảnh trước khi sinh output.
